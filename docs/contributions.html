Contributions

The movie genre classification project was a collaborative effort by a team of two members, Ranjeet Ganeshprasad Rajeshirke and Janhavi Milind Patil.

    Ranjeet Rajeshirke

  • Data Specialist: Acquired TMDb/Kaggle dataset, performed EDA, tokenized text, resized images, created balanced splits.
  • Image Modeling Lead: Fine-tuned ResNet18, applied augmentations (rotations, flips).

    • Janhavi Patil

  • Text Modeling Lead: Designed LSTM with attention, used GloVe embeddings, tuned sequence length.
  • Fusion & Evaluation Lead: Implemented concatenation-based fusion, trained model, evaluated metrics, built website.
