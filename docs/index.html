Multi-Modal Movie Genre Classification

Introduction

Genre classification enhances media analysis and recommendation systems by categorizing movies into genres like Drama, Comedy, or Horror. Using only plot synopses or posters can miss nuanced genre cues, so this project combines both for improved accuracy. Built in PyTorch, it explores data preprocessing, model design, and multi-modal fusion.
