Code Access

GitHub: movie-genre-classification

Setup Guide

1. Create Conda environment:
conda create --name ai-env python=3.9
conda activate ai-env

2. Install dependencies:
conda install pytorch torchvision torchaudio cpuonly -c pytorch
pip install pandas numpy scikit-learn matplotlib seaborn

Data Access

The dataset is sourced from TMDb via Kaggle and preprocessed into train_fixed_updated.json. Follow these steps to obtain and prepare the data:

  1. Download Dataset: Access the dataset from Kaggle (e.g., TMDb 5000 Movie Dataset or similar) at kaggle.com/datasets. Alternatively, download the preprocessed train_fixed_updated.json from the project repository: github.com/username/movie-genre-classification.
  2. Place Data: Save train_fixed_updated.json in the project root directory (e.g., movie-genre-classification/).
  3. Preprocess Data: The dataset includes plot synopses, poster images, and genre labels for 84,094 movies. Run the preprocessing script to oversample the training set (74,000 samples) and generate padded inputs: 
    python preprocess_data.py
    This script tokenizes synopses (100 tokens, GloVe embeddings), resizes posters (224x224), and creates train (74,000), validation (7,981), and test (7,963) splits. Outputs include train_padded.npy, train_labels.npy, etc.
  4. Verify Data: Ensure train_padded.shape is (74000, 100) and train_labels.shape is (74000, 19). Check sample counts for rare genres (e.g., TV Movie: [Insert count, e.g., 350], History: [Insert count, e.g., 420]) using: 
    python -c "import pandas as pd; train_df = pd.read_json('train_fixed_updated.json'); print(sum(train_df['genre_labels'].apply(lambda x: x[15] == 1)))"
