Reproducibility & Instructions

Code Access

GitHub: movie-genre-classification
Main script: FusionModel.ipynb

Setup Guide

1. Create Conda environment:
conda create --name ai-env python=3.9
conda activate ai-env

2. Install dependencies:
conda install pytorch torchvision torchaudio cpuonly -c pytorch
pip install pandas numpy scikit-learn matplotlib seaborn

Data Access

The dataset is sourced from TMDb via Kaggle and preprocessed into train_fixed_updated.json. Follow these steps to obtain and prepare the data:

  1. Download Dataset: Access the dataset from Kaggle (e.g., TMDb 5000 Movie Dataset or similar) at kaggle.com/datasets. Alternatively, download the preprocessed train_fixed_updated.json from the project repository: github.com/username/movie-genre-classification.

Run Commands

Clone repo: git clone https://github.com/RanjeetRajeshirke10/movie-genre-classification.git
Run: python fusion_model_cpu.py > fusion_final_run.log 2>&1
