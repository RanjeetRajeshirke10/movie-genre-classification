Reproducibility & Instructions

Code Access

GitHub: movie-genre-classification
Main script: FusionModel.ipynb

Setup Guide

1. Create environment:

Create a virtual environment first, use the command "python -m venv name_of_env" and then activate the environment with the command "name_of_env\Scripts\activate"


2. Install dependencies:

pip install torch==1.12.0 torchvision==0.13.0 pandas numpy scikit-learn matplotlib seaborn


Data Access

The dataset is sourced from TMDb via Kaggle and preprocessed into train_fixed_updated.json. Follow these steps to obtain and prepare the data:

    Download Dataset: Access the dataset from Kaggle (e.g., TMDb 5000 Movie Dataset or similar) at kaggle.com/datasets. Alternatively, download the preprocessed train_fixed_updated.json from the project repository: github.com/username/movie-genre-classification.

Code Access

All source code is available in the project repository: github.com/username/movie-genre-classification. Key files include:

  • FusionModel.ipynb: Implements the fusion model (LSTM + ResNet18).
  • TextModel.ipynb: Implements the text-only model (LSTM with attention).
  • ImageModel.ipynb: Implements the image-only model (ResNet18).

Clone the repository: 

git clone https://github.com/username/movie-genre-classification.git
cd movie-genre-classification
